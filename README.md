# Scraper-Animextremist

### Usage

```
 usage: scraper_animextremist.py [-h] [-u URL] [-d] [-a]

optional arguments:
  -h, --help         show this help message and exit
  -u URL, --url URL  url del manga a descargar
  -d, --download     descarga un capitulo
  -a, --all          descarga todos los capitulos

```
